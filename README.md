# Advent of Code

My take to the [Advent of Code](https://adventofcode.com/) puzzles.

```bash
# Run the tests
$> v test .

# Compile the runner
$> v -prod aoc.v -o aoc

# Run a single day
$> ./aoc single -y 2020 -d 1

# Run a full year
$> ./aoc year -y 2020
```
